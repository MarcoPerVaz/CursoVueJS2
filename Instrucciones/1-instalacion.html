# Concepto
1 De preferencia usar Laragon por los Hosts virtuales
2 Comando git: git clone https://github.com/MarcoPerVaz/CursoVueJS2.git
3 Lo necesario para usar Vue, se usará directamente en el HTML
4 El menú principal es menu.html
Nota: El archivo menu.html está configurado para ser usado en cualquier máquina, pero de ser necesario usted puede manipular los elementos html href para redirigir a donde se necesite.
Nota: A partir del paso 5 se necesitan pasos adicionales
5 Pasos para poder usar el proyecto Laravel5.4-vue
  • Se copia el archivo .env-example y se pega cambiandole el nombre a .env
  • Se edita el archivo .env (Los datos dependerán de como este configurado su Gestor de Base de Datos y el nombre de su BD)
  • Desde consola (Git Bash) se debe acceder a la ruta del proyecto
  • Para este proyecto fue: /c/laragon/www/CursoVueJS2/Laravel5.4-vue
  • Se usa el comando composer install
  • Se usa el comando php artisan key:generate para generar una llave en el archivo .env
6 Lo necesario para usar Vue, se usará directamente en el HTML
